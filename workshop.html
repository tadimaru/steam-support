🎯 Workshop でできること

Steam Workshop 連携を使うと、マップエディタで作ったオリジナルマップを ワンクリックで公開・共有できます。フォルダを ZIP で受け渡しする必要はもうありません。

  • 投稿 — 自作マップを Steam Workshop にアップロードして一般公開する
  • 購読（ダウンロード） — 他プレイヤーのマップを購読すると自動でダウンロードされ、すぐに遊べる
  • ブラウザ — ゲーム内から Workshop のマップを検索・絞り込み・並び替えして探せる
入口はどこ？ タイトル画面の 「ユーザーマップ」 画面を開くと、そこから Workshop ブラウザを開くボタンがあります。 投稿はマップエディタ側から行います（自作マップを投稿する）。
購読したマップはどこに出る？ 購読・ダウンロードしたマップは、タイトルの「ユーザーマップ」一覧に、自分で作ったマップと並んで表示されます。 起動のしかたは ユーザーマップを起動する と同じです。

🔍 みんなの投稿を探す・遊ぶ

ゲーム内の Workshop ブラウザで、公開されているマップを探して購読できます。

  1. タイトル画面 → 「ユーザーマップ」を開く
  2. Workshop ブラウザを開くボタンを押す（一覧が読み込まれる）
  3. 「みんなの投稿」タブでタグ絞り込み・並び替えを使ってマップを探す
  4. 気になるマップの 「購読してDL」を押すとダウンロードが始まる
  5. ダウンロード完了後、表示が 「購読中（DL済）」 に変わる
  6. ブラウザを閉じ、「ユーザーマップ」一覧から選んで遊ぶ

タブ

  • みんなの投稿 — Workshop に公開されている全マップ。タグ絞り込みと並び替えが使えます
  • 自分の投稿 — あなたが投稿したマップ。非公開（Unlisted）にしたものもここに表示されます

並び替え

「みんなの投稿」タブでは、一覧の並び順を切り替えられます。

並び順内容
人気購読数・評価の高い定番マップを上位に表示
新着投稿された日時が新しい順
急上昇最近になって注目を集めているマップ

タグで絞り込む

画面のタグを選ぶと、条件に合うマップだけに絞り込めます。 同じカテゴリ内で複数選ぶといずれかに合致（OR）、 違うカテゴリをまたいで選ぶとすべてに合致（AND）で絞り込まれます。

購読を解除するには：ブラウザで購読中のマップから「購読解除」を選ぶと、ダウンロードされたデータが削除され、「ユーザーマップ」一覧からも消えます。

📤 自作マップを投稿する

投稿はマップエディタから行います。まず遊べる状態のマップを用意しておきましょう。

  1. マップエディタで投稿したいマップを開く
  2. ファイルメニュー →「バンドル情報を編集…」を選び、バンドル情報ダイアログを開く
  3. タイトル・説明・タグ・サムネイルを設定する（サムネは 自動撮影 も可）
  4. 投稿ガイドラインへの同意にチェックを入れる（初回のみ。次回からは既定で ON → 投稿ガイドライン）
  5. ダイアログ内の 「保存して Steam Workshop に投稿 / 更新」 をクリックする
  6. 保存後に投稿前チェックが自動で走る。問題なければそのまま投稿、警告のみなら確認して続行、エラーがあれば直して再投稿（投稿前チェック）
  7. 投稿が成功すると Steam Workshop にアイテムが作成され、一般公開される
投稿ボタンが押せないとき：投稿は Steam を起動・ログイン中のみ行えます。また、ガイドラインへの同意チェックが入っていないボタンは無効のままです。
タイトル・説明・サムネはどこから来る？ マップに設定済みのバンドル情報（タイトル・作者・説明・プレビュー画像）が、そのまま投稿の初期値として使われます。 バンドル情報の整えかたは マップエディタ入門 → バンドル情報 を参照してください。
投稿は「一般公開」されます。 投稿したマップは検索やブラウザの「みんなの投稿」に表示され、誰でも購読できる状態になります。 公開したくない場合は、投稿後に Steam のアイテムページで公開範囲を変更してください。

📸 サムネイルの自動撮影

投稿マップのサムネイル画像は、投稿前チェックから自動撮影できます。 マップ全体がきれいに収まった 1 枚を、手間なく用意できます。

  1. バンドル情報ダイアログのサムネ欄で 「✨ 自動撮影」 を押す（撮影済みなら「撮り直す」）
  2. 一瞬ゲーム画面に切り替わり、マップ全体が自動でカメラに収まる（列車は写りません）
  3. 撮影が終わると自動的にバンドル情報ダイアログへ戻り、サムネイルに反映される
自前で用意したプレビュー画像（preview.png）をそのまま使うこともできます。自動撮影は「まだサムネがない」「撮り直したい」ときに便利です。
サムネ未設定のまま投稿しようとした場合は、投稿前チェック画面にも 「🖼 サムネを自動撮影して投稿」 が出るので、そこから撮影してそのまま投稿できます。

🏷 タグを選ぶ

投稿マップには、探しやすくするためのタグを付けます。 タグはあらかじめ用意された選択肢からカテゴリ別に選ぶ方式で、自由入力はありません。 表記ゆれが起きず、絞り込みが正確になります。

カテゴリ例
規模単駅 / 小規模 / 中規模 / 大規模 など
難易度やさしい / ふつう / むずかしい など
時間帯朝ラッシュ / 昼 / 夕ラッシュ など
要素折り返し / 入換 / 増解結 など、マップの特徴
言語日本語 / English など、対応言語
マップの内容に合ったタグを選んでおくと、遊びたい人に見つけてもらいやすくなります。

✅ 投稿前チェック

投稿する前に、マップに問題がないかを自動でチェックします。 結果はエラーと警告の 2 段階で表示されます。

種類投稿できる？内容の例
エラー 投稿をブロック（直すまで投稿不可） タイトルなどメタ情報の不足、存在しない要素への参照（幽霊要素）、てこ名の未入力・重複、運用番号の重複 など
警告 確認のうえ続行して投稿可 ダイヤの矛盾（出発＜到着の時刻逆転、同一番線・同時刻の競合）、未検証シナリオのカバレッジ不足 など
警告は「無視してよい」ではありません。 意図した仕様かどうかを一度確認してから続行してください。特にダイヤの時刻逆転・番線競合は、遊ぶ人の混乱につながりやすい項目です。
カバレッジ不足の警告： まだ一度もプレビュー再生していないシナリオは「未検証」として警告が出ます。投稿前に各シナリオを一通り走らせておくと安心です。

🔁 更新・削除

マップを更新する

一度投稿したマップを直したら、同じマップからもう一度「投稿 / 更新」を実行してください。 新しいアイテムが増えるのではなく、既存の Workshop アイテムが上書き更新されます。 購読者には次回同期時に最新版が届きます。

マップを削除する

投稿を取り下げたい場合は、Steam のアイテムページから削除します。 ブラウザの「ページ」ボタンで Steam オーバーレイのアイテムページを開けます（評価・通報）。

テスト投稿の後始末： 動作確認のための投稿は、わかりやすいタイトルにしておき、確認が済んだらアイテムページから削除しておくと、一覧が散らかりません。

⭐ 評価・通報

マップへの評価・コメント・通報は、Steam 標準の仕組みを使います。 ブラウザで各マップの 「ページ」ボタンを押すと、Steam オーバーレイでそのアイテムのページが開きます。

  • いいね評価 — 良かったマップを評価すると、人気・急上昇の並び順に反映されます
  • コメント — 作者への感想や不具合報告を書き込めます
  • 通報（Flag） — ガイドラインに反する内容を見つけたら通報できます
通報された内容は Steam の仕組みに沿って扱われます。安心して使える場を保つため、問題のある投稿を見かけたら通報にご協力ください。

📜 投稿ガイドライン

マップを投稿するには、初回に投稿ガイドラインへの同意が必要です。 安心して使えるコミュニティを保つための、最低限のお願いです。

  • 自作のマップのみを投稿してください
  • 他者の著作物を無断で使用しないでください（画像・音声・名称などを含む）
  • 公序良俗に反する内容、他者を害する内容を投稿しないでください
投稿されたマップ（UGC）の内容について、開発元は検証・サポートを行いません。 投稿・購読したマップに起因するトラブルについて、開発元は責任を負いかねます。信頼できるマップのみを購読してください。

❓ よくある質問

質問回答
投稿したのにブラウザに出てこない 公開直後は Steam 側の反映に少し時間がかかることがあります。少し待ってから「みんなの投稿」を開き直してください。自分の投稿は「自分の投稿」タブで必ず確認できます。
購読したマップが一覧に出ない ダウンロードが完了しているか確認してください。完了後、タイトル → ユーザーマップ画面を開き直すと一覧に合流します。
エラーで投稿できない 投稿前チェックでエラーの内容を確認し、該当箇所（メタ情報・幽霊要素・てこ名・運用番号など）を直してから再投稿してください。
投稿を非公開にしたい Steam のアイテムページから公開範囲を変更できます。ブラウザの「ページ」ボタンからアイテムページを開いてください。
解決しない場合は サポートページ をご覧ください。
