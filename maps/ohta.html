🎯 マップ概要

大田は、東京神奈川急行の2路線が発着する頭端式ターミナル駅です。 線路はホームで行き止まりになっており、到着した列車はすべて同じホームで方向転換して折り返します。 引き上げ線はなく、入換操作も不要。そのかわり、2路線の到着と出発が駅の入口（1か所の分岐器群）に集中するため、 進路の交差をいかに手際よく捌くかが腕の見せどころです。

  • 番線: 1〜4番線（頭端式・島式2面4線）
  • 雪が野線 … 1番線 / 2番線
  • 田万川線 … 3番線 / 4番線
  • 引き上げ線: なし（折り返しはすべてホーム上で行う）
  • 列車: 全列車が各駅停車・3両編成。種別や両数の変化はありません

🔧 進路一覧

進路は発点てこと着点てこの組み合わせで構成されます。 大田の進路はすべて本線進路で、到着用と出発用の2グループに分かれます。 入換進路はありません。

到着進路（場外 → 番線）

発点てこ「1」が雪が野線から、「2」が田万川線からの到着に対応します。

進路名発点 → 着点用途
1A1 → A雪が野線から1番線に進入する進路
1B1 → B雪が野線から2番線に進入する進路
2B2 → B田万川線から2番線に進入する進路（路線をまたぐ渡り）
2C2 → C田万川線から3番線に進入する進路
2D2 → D田万川線から4番線に進入する進路

出発進路（番線 → 場外）

発点てこ「11〜14」が各番線からの出発に対応します。着点は「E」が雪が野線方面、「F」が田万川線方面です。

進路名発点 → 着点用途
11E11 → E1番線から雪が野線へ出発する進路
12E12 → E2番線から雪が野線へ出発する進路
12F12 → F2番線から田万川線へ出発する進路（路線をまたぐ渡り）
13F13 → F3番線から田万川線へ出発する進路
14F14 → F4番線から田万川線へ出発する進路
2番線だけは両方の路線とつながっています。 1番線は雪が野線専用、3・4番線は田万川線専用ですが、2番線は雪が野線・田万川線どちらからも到着でき、どちらへも出発できます。

🔄 ホーム折り返しの流れ

頭端式の大田では、すべての列車が「到着 → ホーム上でそのまま方向転換 → 出発」という流れで動きます。 引き上げ線への入換がないぶん操作はシンプルで、1本あたりの手順は次の3つだけです。

  1. 到着進路を引く

    接近してくる列車の路線（雪が野線なら発点てこ「1」、田万川線なら「2」）と、 発車標で指定された番線の着点てこを操作して、到着進路を開通させます。

  2. 出発進路を引く

    到着した列車は、客扱いをしながらそのまま折り返し列車になります。 折り返し後の列車番号は、到着時の番号に枝番「_1」が付いたものです（例: 2003 → 2003_1）。 発車標で折り返し後の方面を確認し、その番線の発点てこ（11〜14）と方面の着点てこ（E または F）で出発進路を開通させます。

  3. 発車ボタンを押す

    発車時刻になったら、その番線の発車ボタンを押して発車させます。

⚠ 朝ラッシュは折り返しが約3分間隔
朝ラッシュシナリオでは、到着からわずか3分ほどで折り返し発車する列車が続きます。 到着進路を引いた時点で「次はどの方面へ何時何分に出発するか」を発車標で確認しておくと、余裕を持って捌けます。

🔀 2番線の使いこなし（路線またぎ運用）

2番線は両路線とつながる共用番線です。ほとんどの列車は「来た路線へ折り返す」だけですが、 朝ラッシュには雪が野線から到着して田万川線へ出発する路線またぎの運用が存在します。

到着時到着進路出発時出発進路
2003（雪が野線から2番線着）1B2003_1（2番線から田万川線へ）12F

発車標の方面表示をよく確認し、思い込みで「12E（雪が野線行き）」を引かないように注意してください。 なお進路12Fは駅入口の分岐器群を斜めに横断するため、他の多くの進路と競合します。周囲の到着・出発が落ち着いたタイミングで開通させるとスムーズです。

🔔 発車ボタンの見分け方

発車ボタンは番線ごとに1個、計4個あります。路線ごとに色とベルの音が異なります。

番線路線ボタンの色ベルの音
1番線 / 2番線雪が野線ピンク高いベル
3番線 / 4番線田万川線紫低いベル

⚠ よくある間違い・注意点

① 出発進路の引き忘れ

到着した列車は自動では折り返せません。出発進路の開通と発車ボタンの両方がそろって初めて発車できます。 到着を捌くのに追われて出発側を忘れると、あっという間にホームが埋まります。

② 2番線の方面取り違え

2番線からは雪が野線（12E）と田万川線（12F）の両方へ出発できます。 発車標の方面表示を確認してから進路を引いてください。

③ 駅入口での進路競合

2路線の到着・出発がすべて駅入口の分岐器群に集中します。 競合する進路は同時に開通できないため、先に引いた進路が解錠されるのを待つ場面があります。 発車間際の列車の進路を優先し、到着列車は場外で少し待たせる判断も有効です。
