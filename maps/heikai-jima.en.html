🎯 Map Overview

Heikai-jima is a four-track urban station with passing loops that let express trains overtake locals. There are no turnaround operations — trains simply pass through or stop along the main line. This simpler layout makes it a good starting point for new players.

  • Tracks: Track 1 / Track 2 / Track 3 / Track 4
  • Passing loops: Yes (used for express overtakes)

🔧 Route List

A route is defined by a Start Lever and an End Lever. For example, route 1A is opened by Start Lever "1" and End Lever "A".

All routes at Heikai-jima are main-line routes (no shunting routes). Start levers light up yellow.
