Help Guide / User Maps / Steam Workshop

🎯 What Workshop Does

Steam Workshop integration lets you publish and share your original Map Editor creations in one click. No more passing folders around as ZIP files.

  • Publish — upload your own map to Steam Workshop and make it public
  • Subscribe (download) — subscribe to another player's map and it downloads automatically, ready to play
  • Browse — search, filter, and sort Workshop maps from inside the game
Where's the entry point? Open the User Maps screen from the title, and there is a button to open the Workshop browser. Publishing is done from the Map Editor side (Publish Your Map).
Where do subscribed maps show up? Maps you subscribe to and download appear in the title's User Maps list, right alongside the maps you made yourself. You launch them the same way as in Playing User Maps.

🔍 Find & Play Others' Maps

Use the in-game Workshop browser to find and subscribe to published maps.

  1. Title screen → open User Maps
  2. Press the Open Workshop browser button (the list loads)
  3. On the Everyone's posts tab, use tag filters and sorting to find maps
  4. Press Subscribe & DL on a map that catches your eye to start the download
  5. Once done, the label changes to Subscribed (downloaded)
  6. Close the browser and pick the map from the User Maps list to play

Tabs

  • Everyone's posts — all maps published on Workshop. Tag filtering and sorting are available
  • My posts — maps you have published. Ones you set to Unlisted also appear here

Sorting

On the "Everyone's posts" tab, you can change the order of the list.

OrderWhat it shows
PopularEstablished maps with high subscriptions and ratings on top
NewestMost recently published first
TrendingMaps gaining attention lately

Filtering by tag

Pick tags on the screen to narrow the list to matching maps only. Selecting several within the same category matches any of them (OR), while selecting across different categories matches all of them (AND).

To unsubscribe: choose "Unsubscribe" on a subscribed map in the browser; the downloaded data is removed and it disappears from the User Maps list too.

📤 Publish Your Map

Publishing is done from the Map Editor. Have a playable map ready first.

  1. Open the map you want to publish in the Map Editor
  2. Choose File menu → "Edit Bundle Info…" to open the Bundle Info dialog
  3. Set the title, description, tags, and thumbnail (the thumbnail can be auto-captured)
  4. Check the box to agree to the publishing guidelines (first time only; it defaults to ON afterward → Publishing Guidelines)
  5. Click "Save & Publish / Update on Steam Workshop" inside the dialog
  6. A pre-publish check runs automatically after saving. If clean it publishes; if only warnings, confirm and continue; if there are errors, fix them and retry (Pre-publish Check)
  7. On success, an item is created on Steam Workshop and made public
If the publish button won't press: publishing works only while Steam is running and you are logged in. The button also stays disabled until the guidelines agreement box is checked.
Where do the title, description, and thumbnail come from? The bundle info already set on the map (title, author, description, preview image) is used as the initial values for the post. See Map Editor — Bundle Info for how to set that up.
Posts are made "public." A published map appears in search and in the browser's "Everyone's posts," and anyone can subscribe. If you'd rather not have it public, change the visibility on the Steam item page after publishing.

📸 Auto-capturing a Thumbnail

The thumbnail image for a published map can be auto-captured. Get a single shot with the whole map neatly framed, with no fuss.

  1. In the Bundle Info dialog's thumbnail field, press "✨ Auto-capture" (or "Retake" if one already exists)
  2. The view briefly switches to the game and the whole map is fit into the camera automatically (trains are not shown)
  3. When done, it returns to the Bundle Info dialog automatically and the thumbnail is applied
You can also use a preview image you prepared yourself (preview.png) as-is. Auto-capture is handy when you have "no thumbnail yet" or "want to retake it."
If you try to publish without a thumbnail set, the pre-publish check screen also offers "🖼 Auto-capture thumbnail and publish", so you can capture and publish right there.

🏷 Choosing Tags

Add tags to your published map to make it easier to find. Tags are chosen by category from a preset list — there is no free text. This avoids inconsistent wording and keeps filtering accurate.

CategoryExamples
ScaleSingle station / Small / Medium / Large, etc.
DifficultyEasy / Normal / Hard, etc.
Time of dayMorning rush / Daytime / Evening rush, etc.
FeaturesTurnaround / Shunting / Coupling, and other map traits
LanguageJapanese / English, and other supported languages
Choosing tags that match your map's content helps the people who want to play it find it.

✅ Pre-publish Check

Before publishing, the map is checked automatically for problems. Results are shown in two levels: errors and warnings.

LevelCan you publish?Examples
Error Blocks publishing (can't publish until fixed) Missing metadata such as a title, references to elements that don't exist (ghost elements), missing or duplicate lever names, duplicate operation numbers, etc.
Warning Can publish after confirming Timetable contradictions (departure before arrival, same-platform / same-time conflicts), insufficient coverage from unverified scenarios, etc.
A warning is not "safe to ignore." Confirm whether it is intended behavior before continuing. Timetable time reversals and platform conflicts in particular tend to confuse players.
Coverage warnings: A scenario you have never previewed once is flagged as "unverified." Running each scenario through at least once before publishing is the safe move.

🔁 Update & Delete

Updating a map

Once you've fixed a map you already published, run "Publish / Update" again from the same map. Rather than adding a new item, the existing Workshop item is overwritten. Subscribers receive the latest version at their next sync.

Deleting a map

To take a post down, delete it from the Steam item page. The browser's "Page" button opens the item's Steam overlay page (Rate & Report).

Cleaning up test posts: Give test uploads an obvious title, and delete them from the item page once you're done checking — it keeps the list tidy.

⭐ Rate & Report

Rating, commenting, and reporting maps uses Steam's standard mechanisms. Press the "Page" button on a map in the browser to open that item's page in the Steam overlay.

  • Like rating — rating good maps feeds into the Popular and Trending order
  • Comments — leave the author your impressions or bug reports
  • Report (flag) — flag content that violates the guidelines
Reports are handled per Steam's mechanisms. To keep this a place everyone can use with confidence, please report problematic posts when you see them.

📜 Publishing Guidelines

To publish a map, you must agree to the publishing guidelines the first time. These are the minimum asks for keeping the community a place everyone can use with confidence.

  • Publish only maps you made yourself
  • Do not use others' copyrighted works without permission (including images, audio, and names)
  • Do not publish content that is offensive to public order and morals or that harms others
The developer does not validate or support the contents of published maps (UGC). We cannot take responsibility for troubles arising from maps you publish or subscribe to. Please subscribe only to maps you trust.

❓ FAQ

QuestionAnswer
I published but it doesn't show in the browser Right after publishing, it can take Steam a little while to reflect. Wait a moment and reopen "Everyone's posts." You can always find your own posts on the "My posts" tab.
A subscribed map doesn't appear in the list Check that the download has finished. Once done, reopen Title → User Maps and it will join the list.
An error blocks publishing Check the error in the Pre-publish Check, fix the spot in question (metadata, ghost elements, lever names, operation numbers, etc.), and try again.
I want to make a post private You can change the visibility on the Steam item page. Open the item page from the browser's "Page" button.
If this doesn't solve it, see the support page.
